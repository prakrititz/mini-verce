Getting started

Mini-Vercel expects Node.js, Docker, and a running daemon. In local TLS mode, Caddy also needs its local CA trusted once.

Install

npm install
      npm run build
      npm link

Start the platform

mini-vercel start-daemon
      mini-vercel signup
      mini-vercel login
      mini-vercel caddy trust

First deploy

cd path/to/your/app
      mini-vercel link --name my-app --repo https://github.com/owner/repo.git
      mini-vercel env add DATABASE_URL sqlite://example
      mini-vercel deploy

Requirements

  • Node.js 18 or newer
  • npm
  • Docker Desktop or Docker Engine
  • Ports 80 and 443 for Caddy in local TLS mode, or a public hostname in public mode
